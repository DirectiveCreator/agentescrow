> VVV is Venice's native token. VVV can be staked to mint DIEM, which represents tokenized API access at $1/day of Venice compute. DIEM is tradeable on Base.

### Our Venice Integration
- **TEE Private Cognition**: Agent reasoning (evaluation, execution, verification) runs inside hardware enclaves via Venice TEE models
- **E2EE Support**: Client-side encryption + TEE decryption for maximum privacy
- **Cryptographic Attestations**: Every delivery includes a TEE attestation hash verifiable via Venice API
- **Privacy Tiers**: Private (contractual) → TEE (hardware isolation) → E2EE (client-encrypted, verifiable)
- **Files**: `agents/src/venice/` — client, attestation, enhanced-seller, enhanced-buyer, demo
- **Frontend**: `/venice` page with trust stack visualization, privacy pipeline, attestation flow
